
## Cara Menjalankan Aplikasi
1. Clone Repository
```
git clone https://github.com/YudaSaputraa/astrapay-spring-boot-notes-app-be.git
cd astrapay-spring-boot-notes-app-be
```
2. Buka Project di IntelliJ IDEA

- Buka IntelliJ IDEA
- Klik Open
- Pilih folder project
- Tunggu hingga proses indexing & dependency Maven selesai

3. Build & Run
- opsi 1

masuk ke dir:

```
src/main/java/com/astrapay/AstrapayBaseExternal.java
```
lalu run project melalui bawaan IDE

- Opsi 2 (melalui terminal IDE)

jalankan perintah berikut:
```
mvn spring-boot:run
```
